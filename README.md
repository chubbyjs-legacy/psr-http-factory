# psr-http-factory

## Description

Common Http-message factory Interface ([PHP FIG][2] [PSR-17][3]).

## Requirements

 * node: 12
 * [@chubbyjs/psr-http-message][4]: ^1.2.0

## Installation

Through [NPM](https://www.npmjs.com) as [@chubbyjs/psr-http-factory][1].

```sh
npm i @chubbyjs/psr-http-factory@1.1.0
```
